Como funciona em 4 passos
📂
1. Carregar Dicionário
open() + read() + close() lêem o arquivo numbers.dict para a memória heap via malloc()
✅
2. Validar Dicionário
dict_is_valid() verifica linha por linha o formato "número: palavra" com parse_line()
🔢
3. Agrupar Dígitos
write_group() divide o número em grupos de 3 dígitos da esquerda para a direita
📢
4. Imprimir Palavras
write(1, ...) imprime cada palavra com seus separadores diretamente no stdout
